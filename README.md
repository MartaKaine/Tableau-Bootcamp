# 📊 Tableau Data Visualisation Project (Bootcamp)

This project was completed during my **Data Technician Bootcamp** and focuses on analysing datasets using **Tableau** to identify trends, explore patterns, and present insights through interactive dashboards.

The project demonstrates how data visualisation can be used to transform raw datasets into **clear, interactive stories that support data-driven decision making**.

---

## 📁 Project Overview

Using Tableau, I analysed datasets to explore trends in areas such as **health metrics, population data, and music analytics**.

The goal was to practice building **interactive dashboards** that allow users to explore data through filters, visualisations, and dynamic elements.

The final dashboards combine multiple visualisations to present **clear and engaging insights**.

---

## 🛠 Skills Demonstrated

### Interactive Dashboards

Built dashboards that combine multiple worksheets into a single interactive view, allowing users to explore different aspects of the dataset.

Features include:

- Dashboard layout design
- Interactive filtering
- Linked visualisations
- Dynamic data exploration

---

### Tableau Features Used

- **Filters** to allow users to focus on specific countries, years, or categories  
- **Calculated fields** to derive new insights from existing data  
- **Sorting and aggregation** to highlight trends  
- **Colour encoding** to differentiate categories  
- **Dashboard actions** for interactive exploration  

---

## 📊 Data Visualisations Created

A range of chart types were used to present insights effectively:

- **Bar charts** – comparing values across categories  
- **Line charts** – visualising trends over time  
- **Pie charts** – showing distribution of categories  
- **Scatter plots** – exploring relationships between variables  
- **Maps** – visualising geographic data  

Each visualisation was designed to answer a specific analytical question.

---

## 📈 Example Analysis

Examples of insights explored include:

- Life expectancy trends across continents
- Population distribution by gender
- Relationships between health indicators such as BMI and life expectancy
- Patterns in music characteristics and track popularity

These analyses help demonstrate how visualisation tools can support **data exploration and storytelling**.

---

## 📖 Data Storytelling

A key focus of this project was **data storytelling**.

Rather than just creating charts, the dashboards were designed to:

- Highlight important trends
- Allow users to interact with the data
- Provide clear visual explanations of patterns

This approach helps turn complex datasets into **insights that are easy to understand and explore**.

---

## 🚀 Why This Project Matters

Tableau is widely used in data roles to create **interactive dashboards and business intelligence reports**.

This project demonstrates my ability to:

- Transform raw data into meaningful visualisations
- Design clear and engaging dashboards
- Communicate insights through data storytelling

These are essential skills for **Data Technician and Junior Data Analyst roles**.
